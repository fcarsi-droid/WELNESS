# Wellness App

## Setup rápido

1. Suba este repositório no GitHub
2. Importe no Vercel
3. Adicione as variáveis de ambiente no Vercel:
   - DATABASE_URL (do Neon)
   - GOOGLE_CLIENT_ID
   - GOOGLE_CLIENT_SECRET
   - APP_URL (URL gerada pelo Vercel, ex: https://wellness-xyz.vercel.app)
4. Faça deploy

O banco já deve estar criado via SQL Editor do Neon.
